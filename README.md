Этот скомпилированный исходный код взят из официальной версии OpenWRT и был изменен следующим образом: добавлен HomeProxy; теперь доступ к веб-интерфейсу осуществляется принудительно по протоколу HTTPS; изменено имя хоста.

<br/>
本編譯源碼來自官方openwrt并做出以下修改：<br/>
添加了homeproxy <br/>
後臺web訪問強制https<br/>
修改了hostname<br/>

<br/>
Mã nguồn đã biên dịch này được lấy từ OpenWRT chính thức và đã được sửa đổi như sau: Đã thêm HomeProxy. Truy cập web phía máy chủ hiện bắt buộc phải sử dụng HTTPS. Tên máy chủ đã được sửa đổi.

<br/>

이 컴파일된 소스 코드는 공식 OpenWRT에서 가져온 것이며 다음과 같이 수정되었습니다. HomeProxy가 추가되었습니다. 백엔드 웹 액세스가 이제 HTTPS로 강제됩니다. 호스트 이름이 수정되었습니다.

<br/>

Bu kompiliasiýa edilen çeşme kody resmi OpenWRT-den alyndy we aşakdaky ýaly üýtgedildi: HomeProxy goşuldy. Arka tarapdaky web girişi indi HTTPS-e mejbury ýagdaýda ulanylýar. Host ady üýtgedildi.
<br/>

این کد منبع کامپایل شده از OpenWRT رسمی است و به شرح زیر اصلاح شده است: HomeProxy اضافه شده است. دسترسی به وب در قسمت مدیریت اکنون به HTTPS اجباری شده است. نام میزبان اصلاح شده است.
